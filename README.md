# AI Code Reviewer 

It's a web-based tool that uses artificial intelligence to analyze, improve, and understand code in real time. Users can paste their code, select the programming language, and receive a structured analysis that includes error detection, best practices, suggestions, and detailed explanations, as well as automatically generate an optimized version with a diff-style view of the changes.

 ### Technology Stack

- Next.js 14 (App Router) — main framework with server and client components
- TypeScript — static typing throughout the application
- Tailwind CSS — utility styles
- Groq API (Qwen 3 32B model)
- Vercel — deployment and hosting with automatic CI/CD from GitHub
